The sub-district identifier within the district. For pre-confederation censuses, and for the 1906 and 1911 censuses, this is a sequential number within the district. For the 1871 through 1901 censuses it is a letter, combination of letters, or a letter followed by a roman numeral exponent.

This is part of the unique key of the sub-district. Caution must be taken in changing it to ensure that the combination of sub-district and division remains unique within the district.

Where the sub-district was too large to be enumerated by a single enumerator it was divided into multiple divisions or wards.

This is part of the unique key of the sub-district. Caution must be taken in changing it to ensure that the combination of sub-district and division remains unique within the district.

Note that when you click on the "Add Division" button, the new line has this field set to 1 larger than the value of previous row. The application does not check whether this new value is unique. For example if you add a division after division 3 in a sub-district that has 4 divisions the new row will specify division 4, even though that division number is already in use. It is your responsibility to resolve this before clicking on the update button.

The name of the subdistrict. This does not have to be the same in all divisions of the sub-district, although that is normally the case.

The number of pages in the population schedule or schedule of the living. This is a decimal number.

The first page number in the schedule of the living or population schedule. This is a decimal number.

In most cases this is 1. However in the 1851 and 1861 censuses the pages were not consistently numbered by the enumerators, and the pages were subsequently numbered sequentially through the entire sub-district when the original documents were microfilmed (and the originals destroyed!).

The page increment within the division. In a few cases when the 1851 and 1861 census pages were numbered, during the microfilming process, the technician incremented the page number twice for each leaf of the schedule of the living because information about the individuals was recorded on both the recto and the verso of each leaf. In this case the application has to know that only the odd page numbers are used to identify the page[s] containing information on an individual.

The total number of individuals enumerated in this division. This is a decimal number.

The identifier of the microfilm reel at the Library and Archives of Canada (LAC).

The identifier of the microfilm reel in the Family History Library of the Church of Latter Day Saints. This is a decimal number.

The section identifier in the LAC image database. This is used in different ways in the various censuses. Sometimes it identifies a directory, sometimes it is part of the file name. This is a decimal number

The frame number in the LAC image database. This is a decimal number.

The number of frames in the LAC image database for the population census of this division. This is a decimal number.

Any comments or remarks about the location. For example you can use this to more precisely describe the geographic area covered by the sub-district.

This button is used to add a new division into the district. This new division has default values filled in to add it to the associated sub-district. However once inserted you can alter any fields. Note that the first two columns are part of the unique identifier of the division. The remainder of this unique identifier is displayed in the header above the input form. You are responsible for ensuring that the combination of the values of the first two columns remains unique within the census district.

This button is used to delete the current row from the table.

Click on this button to see the detailed information about all of the pages in this enumeration division.
