This directory contains templates used by the web-site to control the presentation of information within web-pages. Each template contains HyperText Markup Language (HTML) tags which specify the function of each piece of information which is used by the Cascading Style Sheets (CSS) to determine the appearance of that information. Using templates separates the management of data in the database from its appearance. For example the same data will be presented differently to a user depending upon the user's preferred language, which is specified by the optional lang=xx parameter, where xx is one of the ISO 639-1 two character identifiers.

Only web-site administrators need access to the contents of this directory.
