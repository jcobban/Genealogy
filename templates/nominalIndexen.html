Create a New Tree Within the Database

Type a name, starting with the surname, a comma, and the given names. As you type the text you have entered is used to select the individuals to list, starting with the first individual whose name begins with the text you have entered. For example if you enter "Smith, Ma" the listing will begin with the first individual with surname Smith and a given name starting with "Ma" and continuing through the Smiths, and any subsequent surnames if required, until 50 names have been displayed.

If you enter more than one value for given name, for example "Mary Jane", then, for the supplied surname, only individuals who match both values match anywhere in the given name. This is useful for popular surnames where, for example, just "Mary" would return more than 50 responses. In this format the supplied values match anywhere in the given name, not just at the beginning. So "Ma Jane" would match "Emma Jane", "Jane Maria", "Janet Margaret", and so on. You can force a match anywhere by supplying two values that overlap. For example "M Mary" will match only individuals with "mary" anywhere in their given name, not just at the beginning.

Use this selection list to choose which family tree database to display, or to start a new family tree database. Each family tree database is used to organize families which are likely to be related to each other, but not to families in other family tree databases. This does not mean that the trees are never related. When looking for links you can always choose to search all of the trees for a match. However by default the application searches only within the last tree that you looked at. The original family tree database is for families of South-Western Ontario.
