Enter the name of the source as you would like it to appear n the menu of sources.
Enter the title of the document that is referenced by this source. The default value for this field is to make it the same as the Name.
Use this selection list to choose the most appropriate class of documentary source.
Select the entry for the name of the author of the source document. If no entry is suitable, enter the name in the "New Author" input field.
If the author of the source document has never been used before use this field to provide the name of the author of the source document.
Enter the name of the publisher of the source document.
This extended text area is used to record key text extracted from the source document.
This extended text area is used to record notes about the source document.
Use this selection list to identify a repository where this source document is available. This may be a web site, a library, or an archive, but is ideally a public resource.
If the document is not available from any existing repository you can create a new repository by entering its name in this field. As soon as you leave this input text field a dialog opens to permit you to enter information about the new repository.
Enter the call number by which the source document is identified within the repository. For physical documents in a library this will typically be a Dewey Decimal Number or a Library of Congress Number. For a web site this field should contain the specific URL for accessing the source document.
Click this button to apply the changes you have made to the description of the source document.
Click this button to open a dialog for managing the pictures associated with the source document.
Click this button to open a dialog for managing the pictures associated with the master source document.
