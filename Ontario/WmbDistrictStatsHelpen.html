Wesleyan Methodist Baptisms District Status: Help

The Wesleyan Methodist Baptisms is a database created by Ida Reed, who has no association with this web site. This page simply provides an alternate user interface to access the data.

The associated panel displays a district level overview of the progress of the transcription effort for the Wesleyan Methodist Baptisms.

For each municipality this page displays:

  1. The name of the area within the district. This is typically the name of a town or township.
  2. the number of individuals identified in baptisms that have been transcribed.
  3. A button to click on to view the individual baptisms presented in alphabetical order by the names of the individuals.
