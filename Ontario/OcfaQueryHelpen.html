Cemetery Finding Aid Query: Help

The Ontario Cemetery Finding Aid is a database created by a volunteer who has no association with this web site. This page simply provides an alternate user interface to access the data.

The associated page presents a form for selecting relevant entries from the Ontario Cemetery Finding Aid.

The input fields permit identifying which entries to display.

Count
The maximum number of matches to return.
County
Selecting an item from this selection list limits the response to entries from cemeteries in that county.
Township
If you have selected a county, this is a selection list of townships from that county. Otherwise it is a text input field. Selecting an entry or entering a township name limits the response to cemeteries in that township.
Cemetery
Entering a value in this field limits the response to entries from cemeteries whose name contains the supplied text.
Name: Given:
Entering a value in this field limits the response to individuals whose given name contains the supplied text.
Family:
Entering a value in this field limits the response to individuals whose surname matches the supplied value, except if the Soundex option is selected.
Soundex:
Clicking on this value causes the search for individuals by surname to match using a rough phonetic algorithm.
Query:
Clicking on the Query button performs the query. Also pressing the "Enter" key while in a text input field performs the query.
Clear Form:
Clicking on the Clear Form button resets all of the input fields to their default values.
Status:
Clicking on the Status button shows a summary of the progress of the transcription effort.
