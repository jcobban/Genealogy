Wesleyan Methodist Baptisms Volume Statistics: Help

The Wesleyan Methodist Baptisms is a database created by Ida Reed, who has no association with this web site. This page simply provides an alternate user interface to access the data.

The baptisms were recorded in a series of books identified by volume number. This page displays the number of pages and the number of baptisms which have been linked to the family tree for each volume.

Clicking on the View button associated with the volume shows a report on each page of the volume .

This page accepts parameters through the URL:

lang The language code which the user prefers. This is a two character ISO 639 language code optionally qualified by a two character ISO 3166 country code. For example: "en" or "es-MX".
