Death Registration Query: Help

The response to a Death Registration Query is a table summarizing the matching records. Each row of the table contains the following fields:

  • A button which provides access to the details of the transcription. If you are signed on and authorized to update the Death Registration table, then this button appears as Update, otherwise it appears as Display.
  • The year of registration of the death.
  • The registration number within the year.
  • The surname and given names of the deceased. If this registration is cited as documentation for the birth of the child in the family tree, then the name is a hyperlink to the record in the family tree database.
  • The date of death.
  • The death place.
  • The age at death.
  • The date of birth. If this was calculated from the age it is enclosed in square brackets.
