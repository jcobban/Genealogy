Cemetery Finding Aid County Status: Help

The Ontario Cemetery Finding Aid is a database created by a volunteer unconnected with this web site. This page simply provides an alternate user interface to access the data.

The associated panel displays a county level overview of the progress of the transcription effort for the Ontario Cemetery Finding Aid.

For each township this page displays the number of individuals identified on gravestones that have been transcribed.
