Cemetery Finding Aid Status: Help

The Cemetery Finding Aid is a database created by a volunteer who has no association with this web site. This page simply provides an alternate user interface to access the data.

The associated panel displays a high-level overview of the progress of the transcription effort for the Cemetery Finding Aid.

For each county this page displays the number of individuals identified in gravestones that have been transcribed. Clicking on the View button associated with the county shows a more detailed breakdown of the transcriptions for that particular county.
