Wesleyan Methodist Baptisms Status: Help

The Wesleyan Methodist Baptisms is a database created by Ida Reed, who has no association with this web site. This page simply provides an alternate user interface to access the data.

The associated panel displays a high-level overview of the progress of the transcription effort for the Wesleyan Methodist Baptisms.

For each district this page displays the number of individuals identified in baptisms that have been transcribed. Clicking on the View button associated with the district shows a more detailed breakdown of the transcriptions for that particular district.
