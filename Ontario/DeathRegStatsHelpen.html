Death Registration Domain Status: Help

The associated panel displays a high-level overview of the progress of the transcription effort for death registrations within a single administration domain.

For each registration year this page displays the number of death registrations that have been transcribed. To be considered as transcribed at least the name of the deceased has been transcribed.

For each registration year this page displays the number of death registrations that have been linked to the family tree. In some cases this may be higher than the number of transcribed registrations if the family tree cites a death registration that has not been transcribed. The number of linked registrations is colour coded to give a quick view of the percentage of transcribed registrations that have been linked.

Clicking on the View button associated with the years shows a more detailed breakdown of the transcriptions for that particular year.
