County Marriage Query Form: Help

This page permits requesting to view or update a set of pre-Confederation Marriage Registrations which consists of annual reports from ministers of religion or other marriage registrars submitted to county registrars in Canada West between 1858 and 1867. These reports contain less information than the post-confederation registrations, and are organized differently.

You can search for marriage registrations using several fields:

  1. There is a selection list of supported administration domains. However the only domain which is known to use the information structure supported by this part of the application is the Canada West pre-Confederation Marriages.
    If this is all you supply then the response is a list of all reports for this administrative domain.
  2. You can limit the response to marriages recorded in a particular bound volume of these records by supplying the volume number.
    If you supply only the domain and volume number then the response is a list of all reports within the volume in order by report number.
  3. Within a particular volume you can limit the response to a particular report by report number. Each report consists of a list of marriages performed by a particular minister of religion or marriage commissioner (Justice of the Peace) within a particular year, since each minister was supposed to submit this report at the end of the year.
    If you supply the domain, volume number, and report number then the response is a list of all marriages within the report in order by item number.
  4. The given names of a spouse. This will return a list of all spouses matching the given names.
  5. The surname of a spouse. This will return a list of all spouses matching the family name.

Clicking on Query button, or pressing the Enter key initiates the report.

Clicking on Clear Form button resets all input fields to their defaul values.

Clicking on Status button displays a report of those volumes of county marriage reports that have been partially transcribed.
