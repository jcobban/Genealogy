Get Counties Record: Help

This table contains records representing counties within a state or province.
Specific records are identified by parameters 'domain', and 'code'
If you specify "&options=1" then the associated township records are included.
You can search for a list of source records by specifying one or more of the following parameters.
domain The identifier of the containing administrative domain, such as a state or province. This is a 2-character ISO country code concatenated to a nationally defined code for the administrative domain.
code Unique identifier for the county.
name A string to match against part of a county name.
startyear The year the county was established.
endyear The year the county was dis-established.
