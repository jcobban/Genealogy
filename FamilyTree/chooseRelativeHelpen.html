Choose Relative: Help

This page is the entry to the relationship calculator tool. It provides the ability to select the other individual within the database for which the family relationship of the main individual is to be calculated.

There is a text input field that permits you to enter a search key for the name of the individual. Enter the surname, and if required, a comma and as much of the given name as required to narrow the selection. This value is used to get to the portion of the database containing the relative that you are looking for.

The selection list contains up to 200 individuals starting with the first individual in alphabetical order after the value entered in the Name field. You can scroll through this selection list using the mouse. When you find the relative that you are looking for, click on the name, and then on the "Select" button to display the relationship calculator report.
