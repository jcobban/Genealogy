List of Pictures Help

The Edit Pictures dialog is displayed when you click on the Edit Pictures button on the edit individual page or the edit marriage page. The Edit Pictures dialog displays a list of the pictures associated with the particular record from which it was invoked.

For each picture the dialog displays:

  • The caption on the picture.
  • The type of file. At the present time only image files are supported, however video, sound, and other file formats may be supported in the future.
  • The date, if known, of picture. This can be used to order the pictures chronologically.
  • A button to open the Edit dialog for the specific picture.
  • A button to delete the specific picture.

At the bottom of the dialog there is a button to close the dialog. The dialog can be closed without losing any information. You can also safely close the dialog using the window frame.

There is a button to reorder the pictures in chronological order of date of picture. It is recommended that you put some date or range of dates in the picture date field to facilitate ordering, and as a hint for searching for documentation of the picture.
