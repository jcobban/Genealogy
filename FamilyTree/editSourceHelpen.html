Edit Source Record Help

The Edit Source Record dialog is displayed when you click on the Edit button on a row of the Sources Master List page or choose the option to "Add New Source" when adding a citation on the edit event dialog. You will not be presented with either option if you are not authorised to edit the family tree database.

The Edit Source Record dialog presents a form permitting you to alter any of the pieces of information that are recorded about a Master Source.

The dialog presents the following:

  • The name of the Source Record as an input field
  • The type of Source Record as a selection list.
  • The title of the Source Record as it is presented in the footnotes section of the individual display page. This is typically the same as, and defaults to, the name of the record.
  • The author of the source document as a text input field.
  • The publisher of the source document as a text input field.
  • Text about the source document as copied from the source itself as an extended text area.
  • Notes about the source document as an extended text area.
  • A selection list of existing repositories holding source documents from which to select a repository containing the source document, preferably a repository where the document can be publically consulted.
  • A text input field to be used if it is necessary to add a new repository. Note that the only information that can be defined about a new repository in this form is the name of the repository. It is necessary to go to the Addresses Master List to define any additional information about the repository, such as its mailing address.
  • The call number of the document within the repository, for example its LC number.

At the bottom of the form there is an Update Source button to apply the changes to the master source record.
