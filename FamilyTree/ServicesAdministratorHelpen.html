Family Tree Services: Help

This page lists the functions that apply to the family tree as a whole.

  • Find an individual in the family tree by name. Most casual users of the family tree will go directly to this web page.
  • Add a new individual who is not related to any existing individual in the family tree. If the new individual is related to an existing individual, then the individual should be added as a child of an existing family, as a spouse of an existing individual, or as a parent of an existing individual. If there is no likelihood of the new individual being connected to anyone in the existing tree, you should consider creating a brand new tree.
  • Find, and if authorized update, locations that are used in the family tree. This table contains every location that is referenced by any event in the family tree.
  • Find, and if authorized update, master source records that are referenced by citations in the family tree. A master source is a resource, whether conventional or electronic, that contains documentary evidence supporting the facts recorded in the tree.
  • Find, and if authorized update, address records that contain detailed information about how to communicate with individuals or institutions connected to the family tree, or as a more detailed substitute for a location record.
  • View, and if authorized update, the list of event types. Each entry in this list describes a type of event or fact that can be recorded in the event table for an individual or family. Making this an editable table permits extending and customizing the information that is recorded in the database, and how it is presented to the users.
  • View, and if authorized update, the list of child statuses. Each entry in this list describes a status that a child has at birth, such as being stillborn. Making this an editable table permits extending and customizing the information that is recorded in the database, and how it is presented to the users.
  • View, and if authorized update, the list of child to parent relationships. Each entry in this list describes a type of relationship that a child can have to a parent, such as being adopted. Making this an editable table permits extending and customizing the information that is recorded in the database, and how it is presented to the users.
