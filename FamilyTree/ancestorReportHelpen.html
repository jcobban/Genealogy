Ancestor Tree: Help

This page provides an outline summary of the ancestors of the selected individual. It is generally opened in a frame beside the main page for an individual.

There is a text input field that permits you to specify how many generations of ancestors to display. Just type a new number and press enter to change the report. The last number you entered is remembered for the next time you view an ancestor tree.

There is a checkbox to specify whether you wish the locations of the birth and death events of each individual to be displayed in the report. This option is remembered for the next time you view an ancestor tree. Click on the checkbox to toggle between the options.

There is a Close button. This closes the window if this page is displayed in its own window, or blanks out the frame if it is displayed in its normal position in a frame next to the display of information about an individual.

Each individual is described as to their precise relationship to the main individual. For example not just "great-grandfather', but rather "father's mother's father". The name, birth date, and death date if each individual are collectively made a hyper-link to the main web page for the particular individual, and the hyperlink is color-coded to represent the gender of the individual, red for women and blue for men (or dark red and dark blue if you have previously visited the web page for the individual).
