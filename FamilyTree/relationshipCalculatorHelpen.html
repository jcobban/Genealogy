Relationship Calculator: Help

This page reports the family relationship between two individuals in the database. It provides links to the main web page for each of the individuals, and their common ancestor. This report is normally displayed in a frame beside the page displaying information about an individual.

There is a Close button. If the report is displayed in its own window, the window is closed. If the report is displayed in its typical position in a frame beside the main page for an individual, the frame is cleared.
