Report on Individuals: Help

This page presents a very basic report on individuals from the family tree database as specified by the report request page.

At present there are no options for modifying the appearance of this report from this page. For most fields the simple value of the requested field from the database record is presented. The only exceptions at present are:

  • Dates are formatted in human readable form.
  • Locations are interpreted from the locations table.
  • The numeric key field, IDIR, is presented as a hyperlink to the standard web page.
  • Gender is interpreted as 'Male', 'Female', or 'Unknown'.
  • Some flags are interpreted as 'true' or 'false'.
