Display/Edit LDS Temple: Help

Every LDS temple that is recorded in the Temples Master List. This way all of the information collected about a particular temple is centralized. This, in particular, facilitates ensuring that particular temples are always identified in the same way.

You get to this web page by clicking on the link on the numeric key field on the Temples Master List web page. If you are authorized the text and other fields on this page permit you to modify the characteristics of the temple. Otherwise the fields are all protected. The following information is displayed:

  • The unique numeric key of the temple record, called the IDTR.
  • The unique 5 character identification code for the temple.
  • The temple name as it appears in reports. The only way to modify the appearance of the name is through this field.
  • The date the temple was opened as a decimal number YYYYMMDD.
  • The date the temple was closed as a decimal number YYYYMMDD.
  • The "Used" flag is used by Legacy.
  • The "Tag1" flag is used by Legacy Family Tree software.
  • The "qsTag" flag is used by Legacy Family Tree software.
  • If you are authorized to update temples click on the Update Temple to apply changes you have made to the fields.
  • The Display Individuals using this Temple button displays a list of individuals and marriages that reference this temple.
