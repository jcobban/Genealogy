Bienvenu: Aide

Lorsqu'un utilisateur se connecte à ce site Web, cette page est appelée pour vérifier s'il existe des messages en attente pour l'utilisateur. S'il y en a, cette page de bienvenue s'affiche avec les messages. Sinon, l'utilisateur est immédiatement redirigé vers la page d'index nominal.

La page contient un ou plusieurs messages d'autres utilisateurs. Le nom de l'utilisateur qui a envoyé le message est affiché avec la date et l'heure du message. Le texte du message est affiché. Cela peut inclure une adresse de courrier électronique si l'expéditeur n'est pas un contributeur inscrit. Après chaque message, il y a deux boutons, l'un pour supprimer le message et l'autre pour y répondre.
