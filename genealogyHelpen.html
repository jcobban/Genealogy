Genealogy: Help

This website provides a broad range of genealogy related services, generally focussing on, but not limited to, south-western Ontario.

  • A family tree of tens of thousands of individuals whose families have passed through south-western Ontario, and in particular Middlesex County. This is not a static family tree but rather a dynamic entity that can be contributed to by anyone interested in these families. Since the facilities to update the tree are provided on the web, users can contribute to the tree from anywhere in the world, using any form of access. There are two specific links in this section of the page:
    1. A link to the top level page which provides a set of management tools for all of the tables making up the family tree database.
    2. A link to the nominal index of the Family Tree. This permits you to search for individuals by name and birth date.
  • Access to the contents of the database in machine readable form, for use by applications running on this and other sites is provided by the pages getRecordXml.php and getRecordJson.php.
  • Pointers to other sources of genealogical information broken down geographically. This site focuses in particular on Canada, but there are also special sections on the United States of America, the United Kingdom, and a collection of sites that charge for information.
  • A collection of user guides for particular issues in genealogy.
  • Links to back issues of the monthly newsletter that is sent by e-mail to the registered contributors.
  • Links to back issues of the Monthly Technical Status Reports, which describe the changes made to the functionality of the web-site each month, as well as a list of bugs that have been corrected.

All pages in this web site include a header and trailer section. These sections display the hierarchy of web pages leading to the current page, so you can quickly move back up the tree toward the top. In addition the trailer section contains a link to send a message to the site administrator regarding the page.

The most important element of the header is the button displayed in the upper right corner of the page. This button tracks your status as a user of the web site. When a user first enters the web site this button displays the text Contribute. Clicking on the button displays a user signon/registration dialog. Identifying yourself to the web-site makes many additional features and services available. Once you have signed on this button changes to display your user name. If you click on the button now it takes you to an account management dialog.
