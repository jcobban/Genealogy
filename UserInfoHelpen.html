Welcome: Help

When a user signs on to this web site, this page is invoked to check to see if there are any outstanding messages for the user. If there are then the Welcome page is displayed along with the messages. Otherwise the user is immediately redirected to the nominal index page.

The content of the page is one or more messages from other users. The name of the user who sent the message is displayed along with the date and time of the message. The text of the message is displayed. This may include an e-mail address if the originator is not a registered contributor. After each message there are two buttons, one to delete the message and the other to reply to it.
