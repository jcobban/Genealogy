Birth Registration Query Response: Help

The response to a Birth Registration Query is a table summarizing the matching records.

The title identifies the province or state where the births were registered. Each row of the table contains the following fields:

  • A button which provides access to the details of the transcription. If you are signed on and authorized to update the Birth Registration table, then this button appears as Update, otherwise it appears as Display.
  • If you are authorized to update this table you are also authorized to Delete existing transcriptions.
  • The year of registration of the birth.
  • The registration number within the year.
  • The surname and given names of the child color coded by the gender of the child. If this registration is cited as documentation for the birth of the child in the family tree, then the name is a hyperlink to the record in the family tree database.
  • The date of birth.
  • The birth place.
