Birth Registration Query: Help

The associated panel permits searching a database of Birth Registrations, focussing on Middlesex County in Ontario, Canada.

The title includes the name of the province or state whose birth registrations are to be searched. This cannot be altered through this dialog. It is determined by the page from which you came to get to this dialog.

Identification: Permits searching for birth registrations by specifying the year of registration and the number assigned by the provincial or state Registrar when the records were indexed and microfilmed in the 1950s. Specifying the registration year and number limits responses to the records with registration number between the specified value, and that number plus the value of Count minus one.

Limit: Permits specifying the maximum number of matching entries to display at a time. The default is 20 at a time.

Registration: County: If specified limits the response to births registered in the specified county.

Registration: Township: If specified limits the response to births registered in the specified city, town, or township. For some counties this item is presented as a selection list.

Name: Given: If specified limits the response to births of individuals whose given name matches the specified pattern. The value is a regular expression. In particular if a simple string is given it matches all names that contain the string. For example 'fred' matches 'frederick', 'freddie', and 'alfred', among other names.

Name: Family: If specified limits the response to births of individuals whose family name matches the specified pattern. The type of search depends upon the value specified here and if the Soundex option is selected. In particular if the value is composed of letters of the alphabet and spaces then only names that match exactly will be included, except if the Soundex option is specified. However if the value contains characters that have a special meaning in regular expressions then a pattern match it performed. For example 'smith$' matches names that end in 'smith'.

Soundex: If this option is checked then the Soundex algorithm is used to make the surname search based on sound rather than an exact match.

Sex: If specified limits the response to births of individuals whose sex matches the specified value.

Birth Date: If specified limits the response to births of individuals whose birth date matches the specified pattern. The value is a regular expression. In particular if a simple string is given it matches all dates that contain the string. For example specifying '1882' will match all dates in 1882.

Birth Place: If specified limits the response to births of individuals whose birth place matches the specified pattern. The value is a regular expression. In particular if a simple string is given it matches all birth places that contain the string. For example 'm.*x' matches places whose name contains the letter 'm' or 'M' and where somewhere later in the place name there is a letter 'x' or 'X', for example 'middlesex', 'msx', and 'mexico', among others.

Father: Name: If specified limits the response to births of individuals whose Father's name matches the specified pattern. The value is a regular expression. In particular if a simple string is given it matches all names that contain the string.

Father: Occupation: If specified limits the response to births of individuals whose Father's occupation matches the specified pattern. The value is a regular expression. In particular if a simple string is given it matches all places that contain the string.

Mother: Name: If specified limits the response to births of individuals whose Mother's name matches the specified pattern. The value is a regular expression. In particular if a simple string is given it matches all names that contain the string.

Mother: Occupation: If specified limits the response to births of individuals whose Mother's occupation matches the specified pattern. The value is a regular expression. In particular if a simple string is given it matches all places that contain the string.

Attendant: If specified limits the response to births of individuals whose attending physician's or midwife's name matches the specified pattern. The value is a regular expression. In particular if a simple string is given it matches all names that contain the string.

Informant: Name: If specified limits the response to births of individuals whose attending informant's name matches the specified pattern. The value is a regular expression. In particular if a simple string is given it matches all names that contain the string.

Relation: If specified limits the response to births of individuals for which the relation of the informant to the deceased matches the specified value.

Query Clicking on this button performs the query. This may also be requested by pressing the Enter key while the input focus is in a text field.

Clear Form Clicking on this button resets the input fields to their default values.

Status Clicking on this link displays a page with a high-level summary of the transcription effort.
