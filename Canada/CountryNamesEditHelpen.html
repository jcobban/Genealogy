Edit Alternate Country Names: Help

This page permits providing or modifying the name of a country in languages other than English. It also provides the ability to specify the definite article used with the name of the country in a sentence and the possessive phrase used for identifying resources associated with the country. In English most country names are "uncountable" nouns, which do not take a definite article.

For each la nguage there is a row in the table containing:

  • A two-character language code, as defined by ISO 649, used by other tables to identify a language. This field is read-only except on a line for adding a new name.
  • The full name of the country in the speciied language.
  • A field to specify the definite article used in the language with the name of the country.
  • A field to specify the possessive phrase used in the language with the name of the country
  • A Delete button to delete the name. Although this removes the row from the display it is necessary to click on the Update Database button to actually update the database.

At the bottom of the page there are two buttons:

  • The Update Database button applies the changes you have entered on this page into the database. Note that pressing the Enter key while editing any input field also performs this function.
  • The Add New Language Name button adds a new row at the bottom of the table, permitting you to add a new language.
