Register: Help

This dialog provides the ability for a client to register as a contributor in order to gain access to extended features of the web site. These features include:

  • Contribute to transcriptions of census, birth, marriage, and death databases.
  • Post blogs related to individual records in the family tree.
  • See private information on individual records in the family tree of which you have been granted shared ownership because you have demonstrated to an existing owner that you have a close relationship.
  • Update individual records in the family tree of which you have been granted ownership. This includes adding new immediate relatives, such as spouses, parents, or children.

You get to this dialog by clicking on the "Register: button in the Sign On Dialog. Initially the dialog looks like:

To register as a user of the web site you must supply:

  • The user identifier by which you wish to be known to the web site. This is up to 64 characters long, however you may prefer to keep this short, as you must enter it each time you sign on to the web site. Your browser may assist you with remembering this. This identifier must be unique within the web site. If the value you supply is not acceptable to the implementation the text will be highlighted in red.
  • The password that you wish to use when you identify yourself to the web site. This is at least 6 characters and at most 32 characters long.
  • Repeat the supplied password.
  • The E-mail address at which you wish to be contacted as part of the registration process. This is up to 256 characters long. You will be able to change this later using the account management web page. If the value you supply is not acceptable to the implementation the text will be highlighted in red.
  • A checkbox where you can choose whether or not to receive regular notices about changes to the web-site at the registered e-mail address.

Then click on the "Register" button at the bottom of the page body. If there is a problem a message is displayed and you are given the opportunity to correct the registration. Otherwise you are directed to respond to the e-mail that will be sent to you to confirm your registration.

Clicking on the "Close" button closes this dialog without registering the user.
