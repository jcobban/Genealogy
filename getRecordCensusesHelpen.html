Get Censuses Record: Help

This table contains records representing censuses that are supported for citations by transcriptions.
Specific records are identified by parameters 'censusid'. All census records are displayed if no parameters are specified.
You can search for a list of censuses by specifying one or more of the following parameters:
name a regular expression for search for a census by name. For example "name=Canada" will match all censuses whose name includes the text "Canada".
linesperpage match censuses with a specific normal number of lines per page.
collective "collective=1" matches virtual censuses that represent a number of censuses enumerated independently at the same time. "collective=0" matches only real censuses.
province match censuses that include the specified province as identified by a code. For example "province=NS" matches censuses that include Nova Scotia.
provinces match censuses which include all of the provinces identified by the supplied value. The province codes must be entered in alphabetical order.
partof match separately administered censuses that are within a country identified by a country code.
countrycode match censuses that are within a country identified by a country code. This includes both censuses administered at the level of that country, and those which are part of a virtual collective census of the country.
