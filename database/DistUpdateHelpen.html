Census District Update Status Help

This page displays the results of a request to update a portion of the Census District table. If the update was successful a Update Census CAyyyy button is presented, which takes you back to the portion of the Census District table that you were just editting.

If anything went wrong you will instead see error messages describing the problem.

Note that there is a hyperlink Specify next census to update in the page header that takes you back to the beginning of the district table edit process.
