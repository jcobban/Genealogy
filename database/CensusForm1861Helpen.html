1861 Census: Transcription Help

This form permits you to:

  • View every field in the transcription of a page of the 1861 Census of Canada even if you are just visiting the site.
  • Transcribe a page of the 1861 Census of Canada.
  • Proofread a page of the transcription.
  • Contribute field level notes.

If you are not signed on with authorization to update the census database then the information on this page is read-only.

Header Information

The complete identification of the page is displayed centered at the top of the form. This is in the form:

CW, dist 99 Countyname, subdist 999 Township Name, div 1 page 9

Where the first two letters are an abbreviation for the name of the colony (future province):

CE
Canada East (future Quebec)

CW
Canada West (future Ontario)

NB
New Brunswick

NS
Nova Scotia

PI
Prince Edward Island

At the left margin, except if you are on the first page of a census division, there is an arrow pointing left. Clicking on this arrow takes you to the preceding page in the division. At the right margin, except if you are on the last page of a census division, there is an arrow pointing right. Clicking on this arrow takes you to the next page in the division.

The next line identifies the transcriber of this page. If the page has not yet been transcribed your user name is inserted here, however you will only become the transcriber for the page if you update the page.

If the page has been transcribed, and someone has volunteered to proofread the page, then the next line identifies the proofreader.

The next line of the form normally contains a Display Original Census Image button. When you click on this button the browser window is split into two halves, and the original census image on the Library and Archives of Canada web site is displayed in the right-hand half of the window. The keyboard shortcut for this button is Alt-I.

It is possible that the URL of the image associated with page in the database is incorrect. In this case you can use the backward and forward links in the image display window to move through the image set for the census until you find the correct image. If you are signed on and authorized to update this page then when you update the census page the associated image URL is corrected so that subsequent visitors will see the correct image.

If you are signed on and authorized to update the database this is followed by a Correct Image URL button. Click on this button to request the option to correct the Image URL if the existing URL recorded for this page is incorrect. The keyboard shortcut for this button is Alt-C.

If you are signed on and authorized to update the database this is followed by a Match Family Tree Citations button. Click on this button to request a search of the family tree database for individuals that have citations to this page. These individuals are compared by name, gender, and age to the individuals on this page, and if there is a good enough match a linkage is established from the corresponding row of the census to the family tree.

Body of the Page

The bulk of the web page consists of a fill-in-the-blanks form which is organized to model the layout of the original image, even to bold horizontal lines after each 5th row of the form. This form contains a text input field for each field in the original form. Most fields have some auto-completion features so you do not have to type the complete value, just start an abbreviation and when you tab out of the field it will be completed. For additional details on how to fill in any field just let the mouse hover over the field.

The columns of this form represent:

  1. The line number on the original form, from 1 to 50. Note that there is an Add a Row button at the bottom of the form to add an extra line if the enumerator squeezed one in.
  2. The family number. Note that the value you enter is replicated into subsequent rows, so you only need to enter this on the first line of a family.
  3. The surname field is used to specify the surname of each individual. This value is replicated into subsequent rows, so you only need to enter this on the head of household. A left square bracket, "[", entered in this field is expanded to "[Blank]" and all of the other fields in this row are set to empty. Do this where the enumerator left a blank line in the middle of a page. Entering "[d" in this field is expanded to "[Delete]" and all of the other fields in this row and all subsequent rows are set to empty. This is done where the original page has fewer than the standard number of entries, in particular on the last page of an enumeration division.
  4. Given names.
  5. Occupation.
  6. Birth Place.
  7. Married in Year indicator. Some enumerators kindly entered the actual year of marriage, although their instructions merely required a check mark.
  8. Religion.
  9. Location. The purpose of this field in the original census was to record the location of a member of the household who was temporarily absent at the time of the census enumeration. As a result this column is almost always empty. Therefore this field is also used as a convenient place to record the address of the household from schedule 2, enclosed in square brackets to indicate that the information was not present in the original image..
  10. Age at next birthday. Note:
    • Unlike the post-Confederation censuses for the 1861 census the field is the age at the next birthday, not the age at the time of enumeration. This may have been intended to avoid having fractional ages.
    • The enumeration took place in late spring, when the muddy roads had dried enough that the enumerator could get around in a carriage. So generally for January through early May the age and birth year add up to 1861, but for late May to December they add up to 1862.
  11. Sex. Specify M or F. In the original census image this is two columns, one with a check for male, and the next with a check for female. The columns are consolidated for consistency with the post-Confederation censuses.
  12. Marital Status. This is M, S or space, or W. In the original census this was two columns, one that contains a M where the individual was married and a second which contain a mark if the individual was widowed. The columns are consolidated for consistency with the post-Confederation censuses.
  13. Coloured. This contains a mark if the individual was of African descent.
  14. Member of the main family. This is two columns in the original image, one for male members and one for female members. Any non-blank character other than 'N' is interpreted as true.
  15. Member of the main family temporarily absent. Any non-blank character other than 'N' is interpreted as true.
  16. Deaf Any non-blank character other than 'N' is interpreted as true.
  17. Blind Any non-blank character other than 'N' is interpreted as true.
  18. Lunatic Any non-blank character other than 'N' is interpreted as true.
  19. Attending school. This is two columns in the original. Most enumerators put a check mark in the same row as the student, but some just put a total number of students by sex in the last row of the family making it ambiguous exactly which children were in school. Any non-blank character other than 'N' is interpreted as true.
  20. Illiterate adult. Any non-blank character other than 'N' is interpreted as true.
  21. Born in previous 12 months. This is two columns in the original. Most enumerators put a check mark in the same row as the child in question, marking the sex information redundant. Any non-blank character other than 'N' is interpreted as true.
  22. Died in previous 12 months. This is two columns in the original. A very few enumerators actually included the deceased in the census of the family, but normally this is simply recorded in the last line of the family. Record the sex, M or F, of the deceased.
  23. Age at time of death.
  24. Cause of death.
  25. Residence type. Typical examples are Brick, Log, Wood (frame), and Shanty.
  26. Number of stories. This is usually 1, 1½, or 2.
  27. Number of families sharing the residence.
  28. Remarks by the original enumerator. Remarks by the transcriber should be identified by square brackets.
  29. Family Tree (FT) linkage column. If this is a Show button then a linkage to the family tree has been entered into the database. Clicking on the Show button opens a new window with the information about the individual as recorded in the family tree. If this is a Find button then clicking on the button performs a search of the family tree for individuals who have names and birth years which indicate they may be the individual enumerated on this line. A dialog is displayed with a list of potential matches to choose from.

Because of the number of columns in this form you may not be able to see all of the columns at the same time. Since not all of the columns may be important to you, you can hide any column by using the mouse to click on the column header cell. This shrinks the column to a minimum width. If you click on the now very narrow column header cell again the column is restored.

Footer Information

There are three buttons at the bottom of the form:

  • Update Database. Clicking on this button applies the changes to the database. Note that the 'U' in the name is underlined as a reminder that the keyboard shortcut alt-U can be used to submit the update, saving the need to scroll the page down to see this button. For compatibility with spread-sheet programs you may also use ctrl-S to submit the update.
  • Reset to Defaults. Clicking on this button resets all of the fields in the form to their default values. For most fields this is blank.
  • Add a Row. If the enumerator squeezed an extra line in at the bottom of the form, sometimes to avoid splitting a family across a page boundary, clicking on this button adds an extra input line to the form.

The complete identification of the page is repeated centered at the bottom of the form.

At the left margin, except if you are on the first page of a census division, there is an arrow pointing left. Clicking on this arrow takes you to the preceding page in the division. At the right margin, except if you are on the last page of a census division, there is an arrow pointing right. Clicking on this arrow takes you to the next page in the division.
