Census Update Help

This page displays the result of a request to update a single page of a census. The same page layout is used for all censuses.

If a problem was encountered then an error mesage is displayed. The most common situation is that your session with the web server has timed out. In this case click on the "Sign On" button in the top right hand corner. The signon dialog is displayed. Once you have signed on that dialog closes and you should be back with the census update status page. Instruct the browser to refresh the page. You will be asked for permission to resubmit the update. When you permit this, the page in the database is updated.

If the update succeeded then you are generally presented with three hyperlinks to choose from:

  1. "Specify next page to update" takes you to the dialog that prompts you for a specific page in the current census to update.
  2. "Update Page n in Current Division" gives you a short-cut to editting the next sequential page in the current division. This hyperlink is not displayed if you just completed editting the last page in the division.
  3. "Display Detailed Status of Division" takes you to a summary display of the transcription progress of the current division. This provides a convenient graphical interface for selecting the next page to edit.
