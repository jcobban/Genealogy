Census Surname Top Index

The Surname Top Index page sumamrizes all of the surnames that appear in the requested portion of the requested census by the first letter of the surname. The displayed table lists each character with which a surname begins and how many individuals in the selected portion of the census have surnames starting with that letter. The letter is itself a link to a page listing all of the surnames starting with that letter.
