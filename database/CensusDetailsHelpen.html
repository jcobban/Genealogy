Census Details Help

This page displays detailed information from the transcription of a single record in the schedule of living individuals in a census. The same format is used for all censuses.

The complete identification of the page in which the specified record is located is displayed centered at the top of the form. This is a string that may be used for citations within a family tree database. There are forward and backward arrows for moving through the transcription one line at a time.

The bulk of the web page consists of a table listing every field in the record with its value. For compactness two fields are displayed on each row of the table.

The page number is a hyper-link to a summary display of all of the records in the page that this record is located in.

The family number is a hyper-link to a summary display of all of the records in the family to which this record belongs.

If the individual is linked to the family tree then there is a hyper-link to the main page of the individual.
