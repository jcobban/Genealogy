Manage Registered Users: Help

This dialog provides the ability for an administrator to manage the table of registered users.

You get to this dialog by clicking on the "Manage Registered Users" link on the Family Tree Services page.

The input field "Pattern" permits the administrator to limit the response to users whose name includes the specified pattern. This is a "regular expression".

The input field "Authorizations" permits the administrator to limit the response to users whose authorization list includes the specified capability. For example specifying "edit" limits the response to users who have permission to update the database.

The input field "E-Mail Address" permits the administrator to limit the response to users whose e-mail address includes the specified pattern. This is a "regular expression".

The "Send Bulk Mail" link opens an e-mail dialog to send a message to all registered users who have accepted to receive bulk notifications.

The "Search" button performs the search specified by the Pattern, Authorizations, and E-Mail Address input fields, and applies any changes made to the value of the Authorizations column of any user.

The remainder of the page is a tabe of up to 20 registered users who match the specied conditions. Each row of this table permits the administrator to:

  • View the user name.
  • Send an individual e-mail to the user.
  • Delete the user.
  • View and alter the authorizations of the user.
  • Delete a user.
  • Reset the password of the user to a new random value, which is automatically e-mailed to the user.
  • If the user has registered but not yet confirmed, the administrator can manually confirm the registration, although the user has the ability to this for their own account.
