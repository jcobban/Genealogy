Home: Help

This website provides information and a broad range of services. Access to the top level page for each category is by a tabbed menu. Clicking on one of the tabs brings the associated top level page to the foreground. The categories are:

  • Contacts: How to get in touch with me.
  • Resumé: Information on my career experience.
  • Tools: Pointers to useful web sites on information technology.
  • Genealogy Services: Extensive family tree research tools.
  • Cobban Family: Information collected about my family as an example of what I can do as a genealogist.

All pages in this web site include a header and trailer section. These sections display the hierarchy of web pages leading to the current page, so you can quickly move back up the tree toward the top. In addition the trailer section contains a link to send an e-mail to the webmaster regarding the page. The most important element of the header is the button displayed in the upper right corner of the page. This button tracks your status as a user of the web site. When a user first enters the web site this button displays the text "Sign On". Clicking on the button displays a user signon/registration dialog. Identifying yourself to the web-site makes many additional features and services available.
